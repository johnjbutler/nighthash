# Nighthash Security - Landing Page

## Project Overview

**Project Type**: React + TypeScript Landing Page  
**Purpose**: High-converting landing page for end-to-end security consulting and attack simulation optimization service  
**Build Status**: ✅ Production ready

## Features Implemented

### Sections
- **Hero Section**: Impactful headline with animated background, stats bar, and CTAs
- **Penetration Testing Services**: 6 comprehensive service cards (Web, Mobile, Cloud, Network, Wireless, Physical)
- **Compliance Testing**: Dashboard-style presentation with 6 compliance frameworks
- **Corporate Training**: 3 training program cards
- **Why Choose Us**: 4 key differentiators with testimonials
- **Contact/CTA**: Email capture form with consultation call-to-action
- **Footer**: Multi-column layout with navigation and contact info

### Design System
- **Dark hacker aesthetic** with cyber-black backgrounds (#0a0a0a)
- **Neon accent colors**: Blue (#4EF0F4), Purple gradient (#E44FFF → #9F6BFF)
- **Typography**: Orbitron (headers), IBM Plex Sans (body)
- **Custom animations**: Pulse glow, scan lines, floating effects
- **Interactive elements**: Hover effects, gradient borders, glow shadows
- **Grid background pattern** for depth
- **Custom scrollbar** with gradient

### Technical Implementation
- React 18 + TypeScript + Vite
- Tailwind CSS with custom color palette
- Custom CSS animations and effects
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Mobile hamburger menu

## Commands

```bash
npm run dev    # Development server
npm run build  # Production build
```
